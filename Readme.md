UI 구현

1. 시작(Init dialog)
- 이름 -> 이름 입력하면 main dialog의 이름창에 저장됨
- 정/수시 -> 정시면 과목별 능력치에서 수학+10, 수시면 플밍+10 할 생각으로 추가해봤음
	(아직 반영되도록 구현하진 않았음

2. 수강신청(Enrollment)
- 학기마다 수강신청할 수 있는 과목이 달라지도록 구현함
- 콤보박스에서 과목을 선택한 후 '선택'버튼을 누르면 장바구니에 추가됨
- 장바구니에 있는 과목을 선택후 '추가'버튼을 누르면 수강신청목록에 추가됨
(현재는 과목을 일일이 선택한후 '추가'를 눌러야 이동되나 향후 한번에 이동하도록 처리할 예정)
- 수강신청목록에서 원하는 과목을 선택후 '과목취소'버튼을 누르면 해당 과목이 목록에서 사리짐(구현예정)
- '신청완료' 버튼을 누르면 메인 다이얼로그의 과목 콤보박스에 추가(구현예정)

3. 메인
-능력치, 과목별 능력치는 우선 각각 100,50으로 설정해 놓았음 (필요에 따라 조정 예정)
